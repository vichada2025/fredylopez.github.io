# Geolocalización para Red Unidos Vichada

Este mini-sitio permite a testigos y líderes compartir su ubicación exacta con un solo clic.

## Cómo subir a GitHub Pages (gratis)

1. Crea una cuenta en https://github.com
2. Crea un nuevo repositorio llamado: `tuusuario.github.io`
3. Sube estos 3 archivos a la rama `main`.
4. Espera 1-2 minutos. Tu sitio estará en:  
   https://tuusuario.github.io

¡Listo! Comparte ese enlace en tus formularios o WhatsApp.
